Making ML systems efficient, reliable, and deployable

I build methods so machine learning models can interact with data more effectively ‚Äî conserving compute, preserving privacy, and operating robustly when data is scarce.

Research directions

  • Data-efficient reasoning. Algorithms and architectures that extrapolate from dramatically less labeled data.
  • Test-time interaction. Non-monolithic models that consult external sources (databases, retrieval, the web) without prohibitive costs.
  • Responsible data use. Efficient auditing and control mechanisms for privacy, fairness and safety with practical accuracy trade-offs.

At Microsoft Research India, I work on efficient grounding for retrieval-augmented models (e.g., FrugalRAG) and evaluation benchmarks like LiveDRBench.
